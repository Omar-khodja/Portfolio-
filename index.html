Project 01

MAKARR

mobile application aimed at boosting citizen engagement by enabling municipalities to run opinion polls and attract investors. The app featured a robust reporting system that allowed users to capture photos, attach GPS locations, and submit issues directly to administrators, streamlining communication and improving responsiveness. I also developed an investment module that empowered investors to subscribe and view municipal projects by city, enhancing transparency and accessibility. To ensure scalability and efficient bug resolution, I applied Clean Architecture with Riverpod’s AsyncValue. On the technical side, I integrated Firebase authentication, real-time database, and cloud storage, successfully supporting over 1,000 concurrent users. This project combined practical civic impact with strong technical execution, showcasing my ability to build scalable, user-centric solutions.

Project 02

AuraFlix

This app is a Hybrid Movie Recommendation System designed for movie enthusiasts who want personalized suggestions while browsing, searching, and rating films. Its core feature is a hybrid recommendation engine that combines collaborative filtering with content-based filtering to deliver more accurate and tailored recommendations. A standout technical highlight is the integration of the TMDB API with Dio, efficiently handling over 10,000 requests, paired with a Node.js backend that manages recommendation logic and secure JWT-based authentication. This blend of advanced algorithms and scalable architecture makes the app both user-friendly and technically robust.

Recipe APP
